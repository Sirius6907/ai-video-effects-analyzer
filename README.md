# 🎬 AI Video Effects Analyzer

A comprehensive web application designed to streamline the process of analyzing, managing, and creating video effects with AI-powered capabilities, specifically tailored for Adobe After Effects (AE) professionals.

## 🚀 Overview

The AI Video Effects Analyzer bridges the gap between raw video footage and professional motion graphics. By leveraging AI, it assists users in understanding video layers, decomposing effects, and generating actionable scripts for Adobe After Effects.

## ✨ Core Features

- **Effect Decomposition**: Deep analysis of video frames to identify specific motion graphics techniques.
- **AE Script Generation**: Automatically generates `.jsx` scripts for Adobe After Effects based on analyzed patterns.
- **Interactive Tutorials**: Step-by-step guidance on recreating complex visual effects.
- **Preset Management**: Save and organize effect configurations for efficient reuse across projects.
- **Workflow Optimization**: Drastically reduces the time spent on manual effect matching.

## 🛠️ Technical Stack

- **Frontend**: Vite + React for a blazing fast, modern development experience.
- **Styling**: Tailwind CSS for responsive and elegant glassmorphism UI.
- **AI Integration**: Custom logic for analyzing visual patterns and translating them into AE-compatible parameters.
- **Tooling**: PostCSS, ESLint, and Vite for a professional-grade build pipeline.

## 🚦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sirius6907/ai-video-effects-analyzer.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Verify Setup**:
   ```bash
   node verify-setup.js
   ```
4. **Run the development server**:
   ```bash
   npm run dev
   ```
