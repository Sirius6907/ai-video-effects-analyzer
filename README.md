# 🎬 AI Video Effects Analyzer

A comprehensive web application designed to streamline the process of analyzing, managing, and creating video effects with AI-powered capabilities, particularly for Adobe After Effects. This tool assists video professionals in understanding video content and generating interactive tutorials.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **Vite** | Frontend tooling and build |
| **React** | UI library for the dashboard |
| **OpenAI API** | Natural language analysis of video effects |
| **Tailwind CSS** | Professional dashboard styling |
| **PostCSS** | Modern CSS processing |

---

## ✨ Key Features

- 🔍 **Effect Analysis**: Intelligent breakdown of complex video effects.
- 📖 **Tutorial Generation**: Automatically creates step-by-step guides for After Effects.
- 📁 **Effect Management**: Central library for storing and categorizing video techniques.
- ⚡ **Real-time Feedback**: Instant AI insights during the creation process.
- 📱 **Mobile Optimized**: Review effect documentation on any device.

---

## 🚀 Getting Started

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Sirius6907/ai-video-effects-analyzer.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up Environment:
   Create a `.env` file and add your `OPENAI_API_KEY`.
4. Start development server:
   ```bash
   npm run dev
   ```
